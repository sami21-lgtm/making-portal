🎓 DIU Student Portal Clone (Frontend Only)
Hey! This is a clean, frontend-only replica of the Daffodil International University (DIU) Student Portal. I built this project to practice and showcase my HTML, CSS, and Vanilla JavaScript skills.

Since this is a frontend project, it doesn't connect to any real database or server. I've used dummy data to make it look and feel like the original portal.

HTML5CSS3JavaScript

✨ What's Inside?
Secure(ish) Login Page: Has a basic validation system. It won't let you in without the correct dummy ID and Password.
Interactive Dashboard: Shows some dummy academic stats (CGPA, Credits) and a mock notice board.
Detailed Profile Page: A nicely structured profile layout with a profile picture, personal info, and contact details. Data loads dynamically via JavaScript.
Responsive Design: Looks perfect on both laptops and mobile phones. The sidebar automatically collapses into a top menu on smaller screens.
Smooth Navigation: Clicking the sidebar menu items smoothly takes you between the Dashboard and Profile pages.
🛠️ Tech Stack
No heavy frameworks, just the basics:

HTML5 (For the skeleton)
CSS3 (Flexbox, Grid, Media Queries for styling)
Vanilla JavaScript (For logic, DOM manipulation, and LocalStorage)
🚀 How to Run This Locally
Don't worry, you don't need to install Node.js or any complex tools. Just follow these simple steps:

Download or Clone the repository to your computer.
Extract the ZIP file (if downloaded).
Go to the project folder.
Double-click on the index.html file.
It will open in your default browser. Boom! You are good to go.
⚠️ Important Note: To see the profile picture, make sure you have an image file named sami.jpg inside the same folder.
